# 登录页 1.0.0

## 项目简介
**登录页**，是基于 [**JEPAAS低代码开发平台**](https://gitee.com/jepaas/jepaas/blob/develop/README.md) 的一套集账号，动态密码，钉钉，微信，QQ，APP多种登录方式的产品。



## 功能特性
- 账号登录
- 动态密码登录
- 钉钉登录
- 微信登录
- QQ登录
- APP扫码登录
- 忘记密码

## 项目结构
```
插件目录：src/vue/modules/login

├─assets        //资源目录
├─components    //组件
│  └─loading    //加载组件
├─js            //工具类
│  └─config.js  //快捷登录配置
├─page          //页面组件
├─styles        //样式
└─index.js      //入口文件
```
